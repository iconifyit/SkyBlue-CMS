IconJar vs ZIP Archives: When to Use Each

Icon Mason supports both IconJar and ZIP formats for icon storage. Here's when to use each format.

IconJar Format

Best for: Mac users who want rich metadata and organization features.

Pros:

  • Rich metadata support (tags, license info, author)
  • IconSet organization within the archive
  • Compatible with the IconJar app for Mac
  • Efficient storage format

Cons:

  • Not natively supported on Windows (without Icon Mason)
  • Requires specialized tools to edit

ZIP Format

Best for: Cross-platform sharing and simple icon collections.

Pros:

  • Universal format – works everywhere
  • Easy to share with anyone
  • Simple structure – just folders and files

Cons:

  • No metadata support (tags, license, etc.)
  • Less organized than IconJar

Recommendation

Use IconJar for your personal library and daily workflow. Export to ZIP when you need to share icons with Windows users or clients who don't have Icon Mason.
