Icon Mason 2.0 Released

We're excited to announce the release of Icon Mason 2.0, a major update to our Adobe Illustrator extension for managing IconJar archives.

What's New

  • Improved Performance: Faster loading and rendering of large icon libraries
  • Enhanced Search: Improved fuzzy search algorithm for more accurate results
  • New Export Options: Export to IconJar, IconSet.app, or ZIP format
  • Better UI: Refreshed interface with improved accessibility
  • Bug Fixes: Numerous stability improvements and bug fixes

Upgrade Now

Existing users can update through the Adobe Exchange marketplace or download the latest ZXP file. The update is free for all existing license holders.
